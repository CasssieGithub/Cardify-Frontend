## Description
Flashcards have been an effective study tool for a long time. Thus, the creation of our app, Cardify. This site was built to help people study using flashcards. This app was built for users of all ages. In this app, you have the ability to create, edit and delete your deck of cards. You could also add and delete individual cards from your deck. This is the **frontend** portion of Cardify app. Follow backend guidlines to initiate application by clicking [here](https://github.com/CasssieGithub/Cardify-Backend) Hope you enjoy!
